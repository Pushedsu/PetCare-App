# Pet-Care App Service Client

반려동물 일정 기록 & SNS 서비스!

## 추가될 기능
-회원 정보 수정

-게시글 찾기

-게시글 옵션 버튼

-이미지 입력
-...

## 구현된 기능
-게시판

-회원 가입

-로그인 & 로그아웃

-캘린더 ( 일정 추가, 마커 표시, 로컬 스토리지 저장 )

-회원 탈퇴

-...
